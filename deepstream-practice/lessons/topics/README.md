# Topics Learning Path

This folder groups the DeepStream apps into topic-based learning tracks.

Use `00_path.md` as the entry point for the full ordering, then open the topic
you want to study next.

## How to use this tree

1. Pick one topic based on the concept you want to learn.
2. Follow the topic path in order.
3. For each future lesson, mirror the structure from `_template/`:
   - `01_guide.md`
   - `02_coding_guide.md`
   - `03_starter.py`
   - `04_extensions.md`
   - `05_reference.py`
4. Keep `beginning/` unchanged as the prerequisite path.
