# Vision Extensions Topic

Day la topic cho cac plugin mo rong ve computer vision:

- `apps/deepstream-opticalflow`
- `apps/deepstream-segmentation`
- `apps/deepstream-segmask`

## What this topic teaches

- Optical flow metadata and visualization
- Segmentation masks and color mapping
- Mask extraction and image save flow
- Post-processing after inference

## Suggested lesson order

1. `01_opticalflow_basics`
2. `02_segmentation_masks`
3. `03_segmask_postprocess`

## Boundary

- Khong dua topic nay ve baseline inference; day la cac nhanh thi giac nang hon.
- Giu focus vao output dang flow/mask thay vi chi bbox detection.
