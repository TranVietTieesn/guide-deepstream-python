# Beginning

Track nhap mon cot loi cua curriculum. Day la noi bat dau neu ban chua quen
GStreamer, DeepStream, batching, metadata, va RTSP basics.

## Learning Model

- Track nay duoc thiet ke de hoc theo thu tu tu de den kho.
- Moi stage chi nen hoc theo dong: `01_guide.md` -> `02_coding_guide.md` ->
  `03_starter.py` -> `04_extensions.md` -> `05_reference.py`.
- Stage `07_config_variants` la stage doc/config lab, nen tien trinh hoc co the
  nhe hon cac stage code-nang.

## Entry Point

- Lo trinh day du nam o `./00_path.md`.
- Neu muon xem nhanh danh sach stage, mo track nay truoc, sau do vao `00_path.md`.

## What This Track Gives You

- Nen tang GStreamer toi thieu.
- Decode chain va streammux single-source.
- Primary infer va batch metadata.
- OSD overlay va multi-source batching.
- RTSP source walkthrough de chuyen tu local input sang live pipeline.
