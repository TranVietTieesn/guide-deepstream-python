# Lesson 05 Extensions

## Co ban

1. Dem object theo `class_id`.
2. In them tong object theo moi frame.
3. Chuyen probe sang diem khac va ghi lai metadata co khac khong.

## Nang hon nhung van trong boundary lesson

1. Map `class_id` sang labels file.
2. In them bbox `left`, `top`, `width`, `height` theo format de doc hon.
3. Thu probe truoc `nvinfer` va giai thich vi sao object meta chua xuat hien.
